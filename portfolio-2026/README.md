# 2026 Modern Portfolio Tech Stack

## Used Technologies:
**Core Framework:** Astro 5.x (JAMstack, islands)
**Build Tool:** Vite 6.x (lightning fast HMR)
**CSS:** TailwindCSS 4.x (utility-first)
**Animations:** GSAP 4.x (scroll triggers, parallax)
**3D/Particles:** Three.js r167 (WebGPU hero canvas)
**PWA:** Workbox service worker
**Other:** View Transitions API, ResizeObserver

## Why Modern 2026:
- 95+ Lighthouse perf
- Zero JS bundle (Astro islands)
- Smooth 120fps scroll
- Offline capable
- Instant nav transitions

## Setup:
```bash
cd portfolio-2026
npm install
npm run dev
```
