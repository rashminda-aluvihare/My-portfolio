# Rashminda Aluvihare — Personal Portfolio & Web Applications

A modern, responsive, and high-performance portfolio web application built with React and Vite. It showcases academic background, technical skills, web development projects, interactive financial engines, and contact information.

## 🌟 Key Features

- **Interactive Hero & Terminal Bio**: Typewriter animation and interactive 3D perspective JSON code snippet.
- **Dark & Light Mode**: Smooth theme toggling with persistent state management (`localStorage`).
- **Smooth Scroll & Animations**: Scroll-reveal animations and glassmorphism visual design system.
- **Featured Projects & Demos**: Showcases full-stack web applications, PHP/MySQL systems, and financial calculator modules.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile displays.

## 🛠️ Tech Stack

- **Frontend**: React 19, HTML5, Vanilla CSS3 (Custom Design System & Tokens)
- **Build Tool**: Vite
- **Icons**: Lucide React
- **Linter**: Oxlint

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation & Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rashminda-aluvihare/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run local development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
